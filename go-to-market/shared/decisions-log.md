# Лог рішень

Короткий статус продуктових і GTM-рішень. Оновлюй після кожного етапу.

## Закрито

| Рішення | Статус | Джерело |
|---------|--------|---------|
| Ядро цінності = бачити фото на столі; клік другорядний | closed | Discovery session |
| Позиціонування vs wallpaper: окремі фото поруч, без зміни фону | closed | Discovery session |
| v1 без слайдшоу / lock screen / хмари | closed | Discovery session |
| USP формулювання (wallpaper-contrast) | closed (чернетка) | Discovery output |
| Порядок GTM: Discovery → Play → ASO → Growth → Audit | closed | prompts framework |

## Відкрито

| Питання | Статус | Нотатки |
|---------|--------|---------|
| Монетизація (free / ads / freemium) | open | Схиляється до ads або freemium; можливий free |
| Ринок / мова listing | open (lean EN) | Пріоритет більший ринок; СНД як можливий особистий канал |
| Головний образ Store (emotion vs aesthetic vs wallpaper-contrast) | open | Хоче A/B або рекомендацію після аналізу |
| Первинний сегмент (emotional vs stylist) | open | Вагання; для ASO треба буде обрати стартовий меседж |
| 1–2 фічі перед v1 | open | Лише якщо критичні для диференціації після конкурентного зрізу |
| Конкурентний зріз photo-widget apps | open | Ще не зроблено руками |
| Proxy-метрики утримання (крім downloads) | open | Рекомендовано в Discovery output |
