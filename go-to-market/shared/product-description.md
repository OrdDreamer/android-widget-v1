## Коротко

Photo Widget — мінімальний Android-додаток, чия єдина мета: показати одне обране фото з галереї на робочому столі у вигляді змінюваного за розміром віджета, з контролем вигляду й поведінки кліку. Кілька віджетів працюють незалежно один від одного.

Офіційний опис: «Показує обране фото з галереї на робочому столі».

## Позиціонування (з Product Discovery + competitor research)

- Шпалери — фон усього столу; Photo Widget — **окремі фото поруч**, без зміни фону.
- USP: «Окремі фото на home screen поруч зі шпалерами — одне фото на віджет, без зміни фону.»
- Ядро цінності: **бачити** важливе фото; клік/редагування — другорядна зручність.
- Elevator pitch: тримати важливі фото окремими віджетами, не міняючи шпалери; кілька незалежних віджетів; без слайдшоу й хмари.
- Store-меседж (старт): **emotion + wallpaper-contrast**; не pure aesthetic і не privacy-only.
- Диференціація vs категорії: одне важливе фото (не slideshow), cover/contain + alignment 3×3, чесний Photo Picker.
- Ризик ASO: exact name “Photo Widget” crowded (ZipoApps); деталі й keywords → [competitor-research.md](competitor-research.md).

## Вимоги

- Android 13+ (API 33), target API 35
- Без повного доступу до сховища — лише Photo Picker

## Що вміє користувач

**У додатку:** pin на стіл (якщо лаунчер підтримує); список активних віджетів (назва / Віджет #N, розмір у клітинках, мініатюра); редагування; скидання конфігу (віджет лишається на столі).

**При додаванні:** системне `APPWIDGET_CONFIGURE` — той самий екран налаштувань.

**Вигляд:** Photo Picker; назва до 40 символів; поворот ±90° (+ EXIF); alignment 3×3; Cover / Contain; форма прямокутник / закруглений / коло; радіус 0…48 dp; клік: без дії / відкрити додаток / відкрити цей віджет. Без фото клік завжди відкриває налаштування цього віджета.

**Розмір:** розтягується; мін. ~110×110 dp; типовий target 2×2.

## Чого немає (v1)

Слайдшоу, кілька фото в одному віджеті, відео, камера, URL-зображення, lock-screen / Glance, видалення віджета з лаунчера з додатку, UI дефолтних налаштувань.

## Технології

Kotlin, Jetpack Compose + Material 3, AppWidgetProvider + RemoteViews, DataStore Preferences, Coil (прев’ю), ExifInterface. Доставка фото у віджет: `setImageViewBitmap` (не URI/FileProvider).

## Аудиторія (гіпотеза)

- Emotional keepers — важливе фото завжди на виду
- Homescreen stylists — кілька фото поруч зі шпалерами
- Ринок: глобальний Play; Store listing EN primary, UA по можливості
