# Photo Widget

Android-додаток з домашнім віджетом, який показує обране фото з галереї на робочому столі.

## Можливості

- Вибір фото через системний **Photo Picker** (Android 13+)
- Налаштування стилю відображення:
  - **Cover** / **Contain**
  - Форма: прямокутник, закруглений прямокутник, коло
  - Регулювання радіусу кутів
- Зміна розміру віджета на робочому столі (лаунчер)
- Кілька незалежних віджетів з різними налаштуваннями
- Налаштування в додатку та при додаванні віджета
- Закріплення віджета на робочий стіл однією кнопкою (якщо лаунчер підтримує)

## Вимоги

- Android 13+ (API 33)
- JDK 17
- Android SDK 35

## Технології

- Kotlin
- Jetpack Compose + Material 3
- Jetpack Glance (App Widget)
- DataStore Preferences
- Coil (прев’ю в додатку)

## Збірка

```bash
./gradlew assembleDebug
```

APK буде тут:

```
app/build/outputs/apk/debug/app-debug.apk
```

### Локальний SDK

Створіть `local.properties` у корені проєкту:

```properties
sdk.dir=/шлях/до/Android/Sdk
```

## Встановлення

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## Як користуватися

1. Відкрийте додаток **Photo Widget**.
2. Оберіть фото та налаштуйте стиль.
3. Натисніть **Закріпити на робочому столі** або додайте віджет вручну через лаунчер.
4. На робочому столі змінюйте розмір віджета пальцями.
5. Для редагування існуючого віджета відкрийте додаток і виберіть його в списку **Активні віджети**.

## Структура проєкту

```
app/src/main/java/com/photowidget/
├── MainActivity.kt              # головний екран
├── WidgetConfigureActivity.kt   # конфігурація при додаванні віджета
├── data/                        # модель і збереження налаштувань
├── widget/                      # Glance-віджет і завантаження зображень
└── ui/                          # спільний UI налаштувань
```
